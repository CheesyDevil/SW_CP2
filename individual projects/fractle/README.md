# Fractle
This code takes your input and draws a Sierpinski Triangle with the detail you specify. It accomplished this by using the turtle library, specifically by utilizing the cloning tutles function and recursion. 

## How to use the Project
1. install turtle and code runner
2. go to run code configuration and set run interminal to true
3. input how many levels you want
4. watch the fractle being drawn

## List of key features
draws a Sierpinski Triangle to your specified degree
uses recursion

## Installation instructions
to run this code you must instal coderunner and set runcode config uration to true
you also need to have downloaded turtle
