# Simple Grade Book
***
![image of running code(screen shot of propperly runningcode)]9(relative path to image in repository)

This project acts as a simple grade book that stores the ID, Grade Level, Grade, and Name of students. It also includes functions to Search students by ID, Grade level or Name. Finally it gives the option to print out a full list of all students and their information, An option to change a student's grade, and a grade summary.

## How to use the Project
***
1. Download coderunner in the extentions tab
2. Tells user how to use the project
3. Including any external libraries that need to be installed

## list of key features
***
Grade summrzation
ID search
Name search
Grade level search
Ability to change grades

***
